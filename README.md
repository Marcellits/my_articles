## SIMPLE RAILS REST API                            

Example of a basic REST API to perform CRUD functions. Uses PostgreSQL by default. Please add your own database and credentials.            

### INSTALL DEPENDENCIES               
$ bundle install            

###RUN MIGRATION            
$ rails db:migrate             

### RUN SERVER           
\$ rails s
